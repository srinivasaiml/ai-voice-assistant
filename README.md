# AI Voice Assistant

This project is an advanced voice-activated AI assistant built with React, TypeScript, and Groq API.  
It supports hands-free voice commands, real-time conversation history, and can open popular websites via voice.

## Features

- Voice recognition and speech synthesis
- Conversational AI responses using Groq
- Stylish UI with animated backgrounds
- Open supported websites by voice command
- Conversation history panel

## Getting Started

1. **Install dependencies:**
   ```
   npm install
   ```

2. **Add your Groq API key:**
   - Create a `.env` file in the project root:
     ```
     VITE_GROQ_API_KEY=your_groq_api_key_here
     ```

3. **Run the development server:**
   ```
   npm run dev
   ```
